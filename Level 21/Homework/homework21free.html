History of Python

Origin of Python

Python is a high-level programming language created by Guido van Rossum. It was first released in 1991 and was designed to be easy to read and write.

  • Created in the late 1980s
  • First released in 1991
  • Named after "Monty Python"

What is Python used for?

  1. Web development
  2. Data science
  3. Artificial intelligence
  4. Game development

Versions

Important Version:

  • Python 1.0 (1994) First official version
  • Python 2.0 (2000) Added list comprehensions
  • Python 2.7 (2010) Last release of Python 2
  • Python 3.0 (2008) Major language improvements
  • Python 3.6+ Faster and more powerful features
